# 如何写一个排版优雅的文章？

```java
//这里是一个代码块
    private fun reflushMarketInfo() {
        val url = "https://fxhapi.feixiaohao.com/public/v1/ticker?limit=1000"
        val get = HttpUtil.get(url)
        if (get == null) {
            return
        }
        var sarray = Gson().fromJson<List<MarketInfo>>(get)
        val timestamp = sarray[0].last_updated * 1000
        val ofEpochSecond = LocalDateTime.ofEpochSecond(timestamp, 0, ZoneOffset.ofHours(8))
        lastUpdateTime = ofEpochSecond.format(pattern)
        markets = sarray.associateBy({ it.symbol }, { it })
    }
```
